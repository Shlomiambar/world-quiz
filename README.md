

Our World - The Full Database | העולם שלנו - המאגר המלא 🌍
An interactive world map learning application by Shlomi Ambar
English Version
"Our World" is a Web App designed to make learning geography an enjoyable, challenging, and accessible experience. The app includes all 195 recognized countries and offers learning tools and a continuous quiz mode.
✨ Key Features
 * 🗺️ Full Database: 195 countries and capitals divided by continents and categories.
 * 🎯 Continuous Quiz: Choose between a Capitals Quiz or a Countries Quiz in a non-stop question sequence.
 * 💡 Smart Hint System: Reveal letter by letter if you get stuck (Note: using a full hint won't grant a success point!).
 * 🏆 High Score Tracking: The system saves your personal best score locally in your browser (Local Storage).
 * ⏱️ Timer: Track your test time and improve your speed.
 * 👨‍🏫 Personal Touch: System messages are phrased respectfully in the plural and accompanied by Shlomi's persona.
🛠️ Technologies
 * Tailwind CSS: For a clean, modern, and responsive design (mobile-friendly).
 * AudioContext API: For high-quality success sounds directly from the browser.
 * JavaScript (Vanilla): Fast logic without the need for heavy libraries.
 * Google Maps Integration: Direct links to maps for every country and capital city.
🚀 How to Run?
The app runs directly from the browser. No installation is required:
 * Download the index.html file.
 * Open it in any browser (Chrome, Edge, Safari).
 * Recommended: On Android/iPhone, select "Add to Home Screen" to use it as a standard app.
גרסה בעברית
"העולם שלנו" היא אפליקציית אינטרנט (Web App) שנוצרה כדי להפוך את לימוד הגאוגרפיה לחוויה מהנה, מאתגרת ונגישה. האפליקציה כוללת את כל 195 המדינות המוכרות בעולם ומציעה כלי לימוד ומבחן רציף.
✨ תכונות עיקריות
 * 🗺️ מאגר מלא: 195 מדינות וערי בירה מחולקות לפי יבשות וקטגוריות.
 * 🎯 מבחן רציף: בחרו בין מבחן בירות למבחן מדינות ברצף שאלות אינסופי.
 * 💡 מערכת רמזים חכמה: קבלו אות אחרי אות אם אתם מתקשים (אבל שימו לב - רמז מלא לא יעניק לכם נקודת הצלחה!).
 * 🏆 שמירת שיאים: המערכת שומרת את שיא ההצלחות האישי שלכם בדפדפן (Local Storage).
 * ⏱️ שעון עצר: עקבו אחרי זמן המבחן שלכם ושפרו את המהירות.
 * 👨‍🏫 ליווי אישי: הודעות המערכת מנוסחות בלשון רבים מכובדת ומלוות בדמותו של שלומי.
🛠️ טכנולוגיות
 * Tailwind CSS: לעיצוב נקי, מודרני ורספונסיבי (מותאם לנייד).
 * AudioContext API: להפקת צלילי הצלחה איכותיים ישירות מהדפדפן.
 * JavaScript (Vanilla): לוגיקה מהירה ללא צורך בספריות כבדות.
 * Google Maps Integration: קישורים ישירים למפות של כל מדינה ועיר בירה.

 * פתחו אותו בכל דפדפן (Chrome, Edge, Safari).
