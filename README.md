# העולם שלנו - המאגר המלא 🌍
**אפליקציית לימוד אינטראקטיבית למפת העולם מאת שלומי אמבר**

"העולם שלנו" היא אפליקציית אינטרנט (Web App) שנוצרה כדי להפוך את לימוד הגאוגרפיה לחוויה מהנה, מאתגרת ונגישה. האפליקציה כוללת את כל 195 המדינות המוכרות בעולם ומציעה כלי לימוד ומבחן רציף.

## ✨ תכונות עיקריות
* 🗺️ **מאגר מלא:** 195 מדינות וערי בירה מחולקות לפי יבשות וקטגוריות.
* 🎯 **מבחן רציף:** בחרו בין מבחן בירות למבחן מדינות ברצף שאלות אינסופי.
* 💡 **מערכת רמזים חכמה:** קבלו אות אחרי אות אם אתם מתקשים (אבל שימו לב - רמז מלא לא יעניק לכם נקודת הצלחה!).
* 🏆 **שמירת שיאים:** המערכת שומרת את שיא ההצלחות האישי שלכם בדפדפן (Local Storage).
* ⏱️ **שעון עצר:** עקבו אחרי זמן המבחן שלכם ושפרו את המהירות.
* 👨‍🏫 **ליווי אישי:** הודעות המערכת מנוסחות בלשון רבים מכובדת ומלוות בדמותו של שלומי.

## 🛠️ טכנולוגיות
* **Tailwind CSS:** לעיצוב נקי, מודרני ורספונסיבי (מותאם לנייד).
* **AudioContext API:** להפקת צלילי הצלחה איכותיים ישירות מהדפדפן.
* **JavaScript (Vanilla):** לוגיקה מהירה ללא צורך בספריות כבדות.
* **Google Maps Integration:** קישורים ישירים למפות של כל מדינה ועיר בירה.

## 🚀 איך מפעילים?
האפליקציה פועלת ישירות מהדפדפן. אין צורך בהתקנה:
1. הורידו את הקובץ `index.html`.
2. פתחו אותו בכל דפדפן (Chrome, Edge, Safari).
3. מומלץ: באנדרואיד/אייפון בחרו "הוספה למסך הבית" כדי להשתמש בזה כאפליקציה רגילה.
